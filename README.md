# Smart Nutrition Application

A modern web-based nutrition tracking app built with React and Vite, designed to help users monitor their daily calorie and macronutrient intake.

## Features

- Daily calorie and macro tracking (protein, carbs, fat, fiber)
- Meal logging with predefined and quick food options
- Weekly nutrition overview with charts
- Goal setting for fitness plans
- Mobile-responsive design simulating a phone app

## Tech Stack

- React
- Vite
- Lucide React (for icons)
- CSS (custom styling)

## Installation

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd smartnutrition-application
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server:
```bash
npm run dev
```

The app will be available at `http://127.0.0.1:5173` (or similar, as per Vite).

To build for production:
```bash
npm run build
```

To preview the production build:
```bash
npm run start
```

## Project Structure

- `index.html`: Main HTML entry point
- `src/main.jsx`: Main React application code
- `src/styles.css`: Custom CSS styles
- `package.json`: Project dependencies and scripts
